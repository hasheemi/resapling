# Resapling 🌱

Website **Resapling** adalah platform donasi dan dokumentasi aksi penghijauan untuk mendukung kelestarian lingkungan.

🔗 **Live Website**: [Resapling](https://resapling-jaya.netlify.app/pages/)

## 📄 Halaman Website

- **`index.html` (Beranda)**  
  Menyambut pengunjung dengan ajakan untuk berpartisipasi dalam program donasi bibit pohon.

- **`about.html` (Tentang Kami)**  
  Berisi visi, misi, latar belakang program Resapling, detail program, dan anggota tim.

- **`update.html` (Update Program)**  
  Menampilkan:

  - Target penanaman yang telah tercapai
  - Leaderboard donatur
  - Dokumentasi aksi penghijauan

- **`login.html` (Login)**  
  Halaman login sederhana untuk pengelolaan konten website.

## 🛠️ Teknologi yang Digunakan

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Tailwind CSS v3.4**
- **AOS (Animate on Scroll) v2.3.2**
- **Boxicons v2.1.4**

## 🎨 Fitur Desain

- Responsive untuk semua ukuran layar
- Warna sejuk dan nuansa minimalis
- Animasi scroll yang halus menggunakan AOS
- Ikon yang ringan dan modern menggunakan Boxicons
- Layout unik dan variatif
